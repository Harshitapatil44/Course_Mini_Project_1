# Real-Time Fraud Detection System

## Overview  
This project implements a real-time fraud detection system using unsupervised machine learning and Generative AI. It detects anomalous financial transactions based on behavioral patterns and provides AI-generated explanations for better interpretability.

## Features  
- Anomaly detection using Isolation Forest  
- Behavior-based feature engineering  
- Interactive visualizations with Plotly  
- GenAI-powered explanations for flagged transactions  
- Interactive dashboard (Gradio / ipywidgets)  
- Automated ETL pipeline and fraud reporting  

## Tech Stack  
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Plotly  
- OpenAI / Gemini API  
- Gradio / ipywidgets  
- Jupyter Notebook  

## Workflow  
1. Data preprocessing (cleaning, transformation)  
2. Feature engineering (transaction behavior patterns)  
3. Anomaly detection using Isolation Forest  
4. Visualization of anomalies  
5. LLM-based explanation generation  
6. Interactive dashboard for analysis  
7. Automated reporting
